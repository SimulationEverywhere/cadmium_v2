Detailed Description

template<typename T>
class cadmium::_Port< T >

DEVS port with typed messages class.

Typed ports can only hold messages of a given data type. NOTE: modelers don't have to deal with objects of the _Port<T> class. They always interface with Port<T> objects.

Template Parameters
Tdata type of the messages that the port can hold.
