Detailed Description

template<typename T>
class cadmium::_BigPort< T >

typed port for big messages.

Messages are stored and passed as shared pointers to constant messages to save memory. NOTE: modelers don't have to deal with the _BigPort<T> class. They always interface with BigPort<T> objects.

Template Parameters
TData type of the big messages stored by the port.
